# Part4修課心得
終於來到這個學期的期末，透過這次github的作業總整理，才真的仔細地去回顧這學期所走過的一步一腳印。  
從一開始的Codesignal到後來開始刷Leetcode，  
再來是各種資料結構與演算法的學習跟作業，還有CS50全英文課程的轟炸XD  
說真的，現在再回頭看看，真的是知識含量濃度爆高的一學期，甚至已經想不起來自己到底是怎麼走過來的了XD  
大概是從大一入學到現在，投入時間最多、最累卻也最有成就感的一堂課了吧ˊˋ  

老師常常跟我們說：「當你覺得頭腦有一點點燒燒的，有一點痛痛的，那就代表他在成長了。」  
記得一開始時，就連看到codesignal的def都被嚇到，甚至不知道該怎麼呼叫它，  
第一堂課老師說很簡單的Linked List，在這堂課之前也從來沒有接觸過，對它根本是完全陌生的狀態，更不用說其它，  
所以從第一次作業開始，每次都要花上至少30~40小時，去debug、去了解其中邏輯，  
幾乎都是在抱怨跟快要崩潰的情緒中度過的XD  
但很慶幸也很感謝，不管是刷Leetcode、或是完成課堂的作業，邊唸邊抱怨的過程中，  
就算頭腦很燒很痛很累很不知所措，我還是硬著頭皮做完了。  
我很清楚地記得那個Leetcode刷得越來越順手的感覺，在大概1個多月的密集練習後，  
對題目的敏感度上升，相關程式語法也越來越熟悉了，  
就像老師說的，「當你手上的工具越來越多，你才會知道你有什麼可以用，  
一直抱怨而不努力，只會原地踏步，最後也只會成為那群少數。」  


真的很謝謝蔡老師能夠來到這裡，在這學期開設這一門資料結構與演算法課程，  
讓我們能夠很有系統很有架構的去重整、加強自己對於程式的相關能力，  
學習解決問題的核心能力，知道該怎麼去釐清自己解題的脈絡，讓自己寫程式時可以更有效率；  
也謝謝助教能夠在這堂課給我們各種方面的幫助，辛苦地替我們改作業，  
設計了抄襲的檢驗制度，敦促我們能夠擁有自己創造自己程式碼的能力。  
最後要謝謝的是我自己，謝謝我沒有半路逃跑，也沒有半路陣亡，咬著牙拖著步伐，  
熬過了無數個睡不好的夜晚，以及讓人煎熬的作業，最後走到了期末的今天。  


一直印象很深刻老師第一堂課時說的一句話：「好好努力，你會在四個月後，看見不一樣的自己。」  
現在，過了四個月，我可以很有自信地大聲說：「我做到了。」  
辛苦了，謝謝一路上一起走來的大家。
